# Трекер тренировок

Простое и современное веб-приложение для отслеживания тренировок.  
Пользователь может создавать, редактировать и удалять тренировки, отслеживать статистику и визуализировать прогресс.

## 🔹 Функционал

- **Добавление тренировок**
- **Редактирование и просмотр тренировок**
- **Удаление тренировок**
- **Сортировка и фильтрация**
- **Статистика:**
  - Количество тренировок
  - Ударный режим (стрик)
  - График активности (HeatMap)
  - График количества тренировок за выбранный период
- **Сохранение данных** в `localStorage`.
- **Адаптивный и интерактивный UI** (Modal, формы, кнопки).

## 🔹 Технологии

- **Frontend:** React, TypeScript, React Router v6
- **Состояние приложения:** Redux Toolkit
- **Визуализация:** Recharts (LineChart, HeatMap)
- **CSS:** Стандартные CSS с модульной структурой
- **Хранение данных:** LocalStorage
- **Утилиты:** кастомные функции для валидации, склонений и подсчета стрика


## 🔹 Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/AleksandrBel15/TrainingTracker
cd TrainingTracker
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите проект:

```bash
npm run dev
```

Проект будет доступен по адресу: http://localhost:5173 (Vite)
